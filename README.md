# Flag Guessing Game

# Description
- This is a game where players test their knowledge on Spanish-speaking flags. The game will show a random flag and answer choices to guess from. After completing the game, the player recieves their final score.

# Features
- Displays random flags after each round
- Score tracking
- Final results after completing the game
- New game button to restart the game

# Tools used
- HTML
- CSS
- JavaScript
- https://www.wikihow.com/Hispanic-Flags

# Challenge/Learned
- A challenge was connecting the JSON to JavaScript to load the questions and images
- I learned how to connect a JSON file to the JavaScript file
